关于

Webmin 是一个基于网页的类Unix服务器系统管理工具，全球安装超过 1,000,000 次（没错，事宝藏！）。有了它，运维快人一步！比如用户，磁盘配额，服务或者配置文件，比如更改，控制开源应用，再比如 BIND DNS Server，管理 Apache HTTP Server， PHP， MySQL， 还有许多许多好东西。

可通过安装可定制的模块来扩展可用性。 除此之外，还有另外两个扩展其功能的项目：

● Virtualmin 是一个强大的，灵活的，最受欢迎的，最全面的 Linux 和 BSD 系统网络托管控制面板，在全球拥有超过 150,000次 安装。它有开源社区支持的版本，以及功能更丰富的Premium版本；
● Usermin 顾名思义，呈现和控制以用户为中心的功能子集，而不是管理员级别的任务。

Webmin 包括 116 个标准模块，并且至少有同样多的第三方模块。

系统要求
Perl 5.10 或更高。

安装
Webmin 可以两种方法安装：

  1. 下载一个预编译包，可用于不同的发行版（CentOS, Fedora, SuSE, Mandriva, Debian, Ubuntu, Solaris 和 其他发行版）。下载页面直达车;
    注：非常建议在你的系统添加源，这样可以自动更新

  2. 下载并解压源码然后运行setup.sh 脚本，无需任何选项，也就是说会直接安装到当前目录。或者使用命令行参数，例如目标目录。
    注：如果你正在安装 Webmin 到Windows 系统上，你必须运行 perl setup.pl 命令。Windows 版能否正常运行取决于许多程序，和可能不属于标准发行版的模块。你需要 process.exe 命令， sc.exe 命令，以及 Win32::Daemon Perl 模块。
